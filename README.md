## QUICKSTART

connect the arduino.  
start testrpc.

```
npm i -g ethereumjs-testrpc
testrpc

// copy one of the privkeys

```

```
npm i
node js/server2.js <address privKey> <string y isDevMode> <string y runArduino>

```

Navigate to `localhost:3000` to trigger the block chain event again.
